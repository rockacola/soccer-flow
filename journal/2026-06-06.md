# 2026-06-06

## What I worked on

Phase 4 polish and unit test coverage. Fixed a dependency conflict, improved tab navigation behaviour, extracted shared header components, built out the activity log player display, and created a shared `JerseyBadge` component used across three screens.

## Decisions made

- **Tab navigation reset**: used the `isFocused` guard so tapping the active tab pops to the list, but switching tabs preserves stack state. Mirrors the web "home link" convention without being aggressive about it.
- **Own goal**: decided against a dedicated own-goal feature. Record it as a goal with no scorer and add a note if needed. Keeps the data model simple.
- **JerseyBadge border**: added `borderWidth: 1` to the shared badge rather than changing the screen background. The badge is now visible on any background without conditional styling.

## What I learned

`@expo/vector-icons@15.x` requires `expo-font>=14`, which is SDK 55+. Pinning to `^14.0.0` keeps it compatible with SDK 54.

## Problems / friction

Briefly implemented own goal as a dedicated feature, then reverted — the extra complexity in types, service, modal, and activity log wasn't worth it for a label that a note can cover.
