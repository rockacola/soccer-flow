## What I worked on

Added Montserrat as the app-wide custom font (100 Thin, 400 Regular, 500 Medium, 600 SemiBold, 700 Bold). Applied across all 24 StyleSheet files.

## Decisions made

Used `@expo-google-fonts/montserrat` (bundles .ttf files in the JS bundle, no native linking needed). Each weight is a separately named font family (`Montserrat_600SemiBold` etc.), so `fontWeight` is replaced with `fontFamily` everywhere ‚Äî setting both is redundant and can misbehave on Android. The live-match timer keeps its monospace `fontFamily` (Menlo/monospace) for numeric readability. Font loading returns `null` during the brief load window; the splash screen covers this in production builds. Added `fonts` constant to `src/constants/typography.ts` alongside `typeScale`.
